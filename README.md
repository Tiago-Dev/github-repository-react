# Github Explorer

:octocat: Add a Github repository and get yours details and issues.

### [Live Demo](https://5ee81e5f8fc8d38e25e29471--priceless-booth-709baf.netlify.app/)

## Installation

Use the package manager yarn to install.

```bash
yarn install
```

## Start

```bash
yarn start
```
